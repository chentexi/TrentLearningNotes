# 2.1 Git常用命令

| 命令                                    |      作用      |
| :-------------------------------------- | :------------: |
| git config global user.name 用户名 设置 |  设置用户签名  |
| git config global user.email 邮箱       |  设置用户签名  |
| git init                                |  初始化本地库  |
| git status                              | 查看本地库状态 |
| git add 文件名                          |  添加到暂存区  |
| git commit -m "日志信息" 文件名         |  提交到本地库  |
| git reflog                              |  查看历史记录  |
| git reset --hard 版本号                 |    版本穿梭    |
| git log                                 | 详细的历史记录 |

## 2.2.1 Git编辑

 1. 编辑vim  (vim 文件名)

    vim Notes/Git.md 

 2. 编辑退出

    按 ESC，左下角就可以进行输入

    `:w` 保存但不退出

    `:wq` 保存并退出

    `:q` 退出

    `:q!` 强制退出，不保存

    `:e!` 放弃所有修改，从上次保存文件开始再编辑命令历史
